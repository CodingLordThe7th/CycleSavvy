How to install GPX routes

This app supports three ways to add GPX routes:

  1. Preloaded routes (hosted web): Copy GPX files into the preloaded_routes/ folder of the project and reference them in index.html. Update service-worker.js if you use offline caching.
  2. Upload via Profile: Sign in, open Profile, and use the Upload GPX Route control. Uploaded routes are stored per-user and will appear under Saved Routes.
  3. Native/Capacitor app: When you package the app natively you can bundle GPX files with the app or let users import from device storage. Use the Filesystem / File Picker APIs in Capacitor for this flow.

Quick tips:

  • Use readable filenames like iron_horse_trail.gpx (lowercase, underscores).
  • If the route doesn't appear, check the browser Console (Network) and ensure the file path matches exactly.
  • To make uploaded routes searchable or sortable, compute and store metadata (length, bounding box) on upload.

See the full instructions and commands in the repo: README_GPX_INSTALL.md
